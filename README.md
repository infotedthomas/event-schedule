# 3-Day Event Schedule

A beautiful, responsive, and editable event schedule webpage for a 3-day training event.

## Features

- **Password Protected** — Secure access with client-side password protection
- **Fully Editable** — Click any field to edit times, durations, speakers, and topics
- **Responsive Design** — Works perfectly on desktop, tablet, and mobile devices
- **Print-Ready** — Optimized for printing with professional formatting
- **Three-Day Schedule** — Separate views for Friday, Saturday, and Sunday
- **Color-Coded Sessions** — Morning (gold) and afternoon (teal) sessions clearly distinguished

## Password

The default password is: `CUTTER`

## Usage

Simply open `index.html` in any web browser. Enter the password to unlock and view the schedule.

## Editing

Click on any field in the schedule to edit:
- **Time** — Start time of the session
- **Duration** — Length of the session
- **Speaker** — Name of the presenter
- **Topic** — Session title or description

Changes are saved automatically in your browser session.

## Printing

Use your browser's print function (Ctrl+P or Cmd+P) to print the schedule. The layout is optimized for printing.
